# Análisis de Datos - Prosperidad Social (UNAD)

Este repositorio contiene el desarrollo técnico para el análisis de Big Data del programa "Empleo para la Prosperidad" enfocado en el municipio de Facatativá.

## Estructura de la Solución
* **Procesamiento Batch:** Limpieza y agregación de datos históricos desde archivos CSV.
* **Procesamiento Streaming:** Ingesta de datos en tiempo real mediante JSON y Unity Catalog Volumes.

## Requisitos de Ejecución
* Entorno de **Databricks** (Community o Pro).
* Configuración de **Unity Catalog** para la sección de Streaming.
* Dataset oficial de Prosperidad Social.
